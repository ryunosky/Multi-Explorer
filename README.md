## Multi-Explorer

This software allows you to operate up to 4 Explorers in the same screen!

## DEMO

* Size can be changed.
* Switchable between 2 and 4 screens.
  * To 2 screens: Double-click on the center border.
  * To 4 screens: Double-click on the bottom border.
